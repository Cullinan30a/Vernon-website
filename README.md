- 推動培訓文化與人才發展，歡迎有志之士加入

## 開發與部署

### 開發模式

```bash
npm install
npm run dev
# 或
pnpm install
pnpm dev
```

### 生產構建

```bash
npm run build
# 或
pnpm build
```

### 預覽生產版本

```bash
npm run preview
# 或
pnpm preview
```

### 網站部署方式

- **平台**：Vercel + Cloudflare
- **GitHub**：Push 更新後會自動觸發 Vercel 重建
- **DNS 設定**：
  - A 記錄：76.76.21.21
  - CNAME（www）：cname.vercel-dns.com
  - Proxy 狀態：DNS only
