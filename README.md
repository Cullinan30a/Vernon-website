## 📦 安裝和運行

### 環境要求
- Node.js 18+ 
- npm 或 pnpm

### 安裝依賴
```bash
npm install
# 或
pnpm install
```

### 開發模式
```bash
npm run dev
# 或
pnpm dev
```

### 構建生產版本
```bash
npm run build
# 或
pnpm build
```

### 預覽生產版本
```bash
npm run preview
# 或
pnpm preview
```

## 🌐 部署

### Vercel部署 (推薦)
1. 將代碼推送到GitHub倉庫
2. 在Vercel中連接GitHub倉庫
3. 選擇框架預設：React
4. 部署設置會自動配置

### 其他部署平台
- **Netlify** - 支援自動部署
- **GitHub Pages** - 免費靜態網站託管
- **Firebase Hosting** - Google雲端託管
